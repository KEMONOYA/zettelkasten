# Compilation system

A set of programs that perform the *four phases* involved in translating source code files into executable object files. 

These four phases, in order, are:
1. [[Preprocessing phase]]
2. [[Compilation phase]]
3. [[Assembly phase]]
4. [[Linking phase]]
